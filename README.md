# af-magic

Oh My Zsh's `af-magic` prompt, but for Oh My Fish üêü

![screenshot of af-magic for oh-my-fish](./screenshot.png)

## Install

```bash
omf cd
cd themes
git clone https://github.com/martpie/af-magic.git ./af-magic
omf theme af-magic
```

## Update

```bash
omf cd
cd themes/af-magic
git pull
omf theme af-magic
```
